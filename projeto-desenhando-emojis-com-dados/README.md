## 📋 Sobre o Projeto
Este repositório é um portfólio de projetos desenvolvidos para a disciplina de **Análise e Desenvolvimento de Sistemas**. O objetivo central foi explorar como a lógica de programação em Python pode ser aplicada para resolver problemas do mundo real, simular ambientes físicos e criar ferramentas criativas[cite: 13, 14, 15].

O projeto abrange desde sistemas de automação comercial e logística até algoritmos de navegação espacial e integração de matrizes para fins artísticos[cite: 9, 14, 15].

## 🚀 Funcionalidades Principais

### 📦 Gestão de Inventário e Logística
*   **Controle de Estoque**: Gerenciamento de entrada/saída de mercadorias com verificação de saldo em tempo real[cite: 9, 11].
*   **Automação Comercial**: Validação de vendas e módulo de cálculo de troco otimizado por cédulas[cite: 7].
*   **Logística de Distribuição**: Cálculo de rotas baseado em distância e monitoramento de transporte[cite: 4, 6].

### 🏃 Simulação de Segurança e Evacuação
*   **Navegação Espacial**: Algoritmo que mapeia obstáculos físicos (escadas, portas trancadas) em lógica condicional aninhada[cite: 13, 14].
*   **Gestão de Energia**: Implementação de camadas de realidade onde o agente possui recursos limitados, forçando a busca pelo caminho mais eficiente[cite: 14].

### 🎨 Desafios Criativos e Integração
*   **O Criador de Emojis**: Lógica aplicada para geração e manipulação de elementos visuais.
*   **Matrizes Musicais**: Uso de estruturas de matrizes para organização e processamento de dados sonoros ou rítmicos.
*   **O Integrador**: Módulo focado na união de diferentes fontes de dados e lógicas em um único fluxo de execução[cite: 15].

## 🧠 Reflexão Crítica (Aprendizados de ADS)
O desenvolvimento destes módulos permitiu uma evolução significativa na compreensão da lógica computacional[cite: 14]:
*   **Tradução da Realidade**: O processo de "quebrar a cabeça" para transformar um ambiente físico ou um problema de negócios em um modelo matemático funcional[cite: 14].
*   **Eficiência e Rigor**: A percepção de que programar é filtrar o essencial. Um erro de indentação ou um `if` mal planejado pode prender o agente em um loop ou causar erros financeiros no estoque[cite: 14].
*   **Planejamento sob Pressão**: Cada linha de código atua como um plano de segurança para prever falhas e garantir que o objetivo final seja alcançado com o menor gasto de recursos possível[cite: 14].

## 🛠️ Detalhes Técnicos
*   **Linguagem**: Python 3.10[cite: 4, 13, 15].
*   **Práticas Aplicadas**: Tipagem rigorosa com `int()` e `float()` para evitar erros de concatenação e uso de `range(início, fim + 1)` para controle total de ciclos[cite: 12].
*   **Estruturas**: Dicionários para inventário, matrizes para desafios criativos e condicionais aninhadas para segurança espacial[cite: 7, 15, 14].
